2产品与服务

百科词条

词条名
词条是百度百科所含内容的基础分割单位有一个单一的主题用于阐述一件事物一个人物或他们具备特定主题的组合例如茉莉花刘德华2008 年北京奥运会
词条名是由中英文数字或特定符号等组成的具有特定内涵和外延的词语[47-48]   
关于多义词和同义词可以阅读参考资料[49-50]   
词条内容
百度百科所含内容的基础分割单位是词条一个词条由以下若干部分组成百科名片词条正文正文图片与图册地图 词条内链参考资料开放分类相关词条扩展阅读等[51] 
收录内容
百度百科收录的内容包括具体事物知名人物抽象概念文学著作热点事件汉语字词或特定主题的组合例如花中国百子论文唯物主义2008年北京奥运会一般而言词条需有一个单一的主题若多个事物有相同的名称重名事物需在以该名称命名的词条中分不同段落阐释当事物存在标准中文名称时百度百科不鼓励创建以外文单词或词组为名的词条
百科词条内容编辑原则可以参考[52] 

编审系统

用户提交的词条编辑或创建版本根据百科规则由百科编审系统给出处理意见符合百科规则的版本将会更新至词条页予以展示不符合百科规则的版本将由编审系统打回并反馈违规所在
每日23时至次日8时为特殊编辑时段此时段用户提交的版本将延时处理[53] 

用户投诉

百度投诉中心的百科投诉中心是受理关于百度百科的各种投诉的主要场所当用户发现某词条内容触犯法律或词条被恶意修改时可以向投诉中心申请删除相关词条的修改版本当用户修改的词条未获通过而用户对拒绝通过理由有疑议时可向投诉中心申请复议此外投诉中心也负责收集用户对百科提出的各种意见和建议[54] 
百科管理员每日定期集中处理各种投诉并通过百度短消息及时告知用户投诉处理结果

用户积分体系

综述
百度用户在百度百科上参与编辑即可获得奖励积分积分分为经验值和财富值两部分财富值可用于在商城兑换虚拟特权徽章和实物礼品经验值与等级头衔相关[55] 
经验值
经验值是用户在创建修改完善百科词条中获得的一种积分影响用户在百科上的等级用户的经验值增加到一定值时可以晋级并获得更高的头衔从而获得额外的编辑权限当用户的头衔达到六级将可以获得一个在百度Hi建群的权限等级越高获得的建群数越多[55] 
财富值
财富值是根据用户在百科中的优质贡献来获得财富值可以在百科商城专区兑换虚拟实物礼品[55] 
百科头衔
每个用户都会有一套属于自己的积分等级和头衔可以通过创建词条等操作不断增加积分同时等级和头衔也会随积分不断晋升[55] 

3产品文化

优质词条

图片优质初级优质中级优质和高级优质都是百科优质词条的范畴优质词条是百科词条的典范之作具有较强的知识性和专业性且用词规范排版美观阐释适度通俗易懂符合互联网阅读习惯
初级优质和高级优质的评选都以中级优质评选规则为基础进行向下和向上的扩展初级优质的设立是为了中级和高级优质提供进阶通道引导编辑者了解百科编辑规则进而将词条质量提升到中级乃至高级优质大体满足中级优质评选规则要求的就能获得初级优质高级优质是为了将不同编辑难度的优质贡献区分开来所以相比之下高级优质对词条来源的全面性提出了更高要求
图片优质是为了鼓励编辑者完善词条图片信息而设立的与初级中级高级优质相比它的评选表彰对文字的要求较弱对于图片的数量和质量要求更高[56] 

百科之星

百科之星是百度百科用户的一项特殊荣誉颁发给对百科内容建设做出突出贡献努力帮助百科词条质量不断提高的用户百科之星每周产生一位用户可以去百科之星吧自荐或者推荐别人[57] 2007年1月推出[58] 

荣誉徽章

用户参与百科任务等一些百度百科的活动后可以获得虚拟的徽章徽章的图标获得时间及获得原因会在个人中心的荣誉殿堂栏目中标明向访问者展示用户为百度百科作出的贡献[59] 

相关活动

特色活动
百科任务是百度百科为了提高词条内容质量而创办的长期栏目为旧百科首页中的组成版块之一后来下线百科任务发起的权限仅限于百科官方或百科蝌蚪团成员但所有用户均可参与任务相关词条的编辑从而获得额外积分例如百度百科57期任务仰望星空[60] 百度百科63期任务西游记[61]

精彩推荐

百科精彩推荐是充实百度百科首页的模块之一用户可向官方申请推荐词条通过后即可在首页展示以便向更多的用户展示相关词条的精彩内容

4参与团队

核心用户

  百科核心用户形象[72]

百科核心用户是从百度百科用户中产生的优秀科友需具备原创及信息整理能力了解百科相关规则能够为百科词条内容建设持续发光发热
有一定编辑能力而且一直在坚持为百科做贡献的科友就有希望成为核心用户只要百科等级达到四级通过率达到85%复杂编辑达到50个就有资格申请加入百科核心用户体系成为百科核心用户[72] 2010年10月推出[73] 

分类管理员

百科分类管理员是从百科用户中选拔出来的优秀网友主要在百科分类频道页进行编辑推荐工作如果对百度百科中某一分类或某一领域有兴趣并自愿承担维护该分类的责任都可申请进入分类管理员团队[74] 2009年12月29日推出[75] 

蝌蚪团

百度蝌蚪团蝌蚪形象(2张)
百度百科蝌蚪团是由百度百科的热心科友所组成的团队是一批有知识有能力以分享知识协作编辑为共同信仰的科友是百科科友团体的中坚力量
蝌蚪团成员对于词条有较高程度的认知和了解具有较强的词条编辑能力评审及判断能力成员可以通过蝌蚪团工作室协助官方完成建设性工作官方会重点听取他们关于百科发展的讨论与建议同时蝌蚪团成员也是普通科友与官方沟通的桥梁
2007年2月6日推出[76-78]    

知识先锋

知识先锋百度百科校园计划是百度百科与全国重点高校合作开展的社会实践互动项目通过这一项目共同对中文百科知识进行最大规模的传播和分享[79] 2010年12月百度百科知识先锋校园计划启动[80] 

百科学术委员会

百科学术委员会由一群具有社会影响力的行业精英学术权威组成旨在匡正去伪在提供各领域内容的同时对百度百科内容提供权威的解释和专业的判定[81] 2012年09月推出[82] 

百科编修院

百科编修院原名百科质量委员会是由一批具有一定编辑经验和技巧的科友组成的团队为了让人类更加平等地认知世界百度百科正在向更可读更可信的方向发展现招募具有一定百科编辑能力的科友在已有词条内容的基础上按照百科化标准提升词条质量编辑出更可读更可信的优秀词条[83] 2012年11月26日推出2013年5月22日正式更名[84] 

权威合作

百度百科权威合作是百度百科与相关权威机构合作对特定词条进行编辑完善的活动旨在给用户带来更加专业权威可信的信息截至2012年12月3日已有合作机构96家合作编辑词条数12034个[85-86]   

5外界评价
