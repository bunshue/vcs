SimpleGUICS2Pygame
It’s a standard Python 2 and 3 package that replace the special CodeSkulptor modules (simplegui, simpleplot and numeric).
  • Online HTML documentation
    (Complete installation on Window$ in few steps)
  • Complete sources and installers on Bitbucket

The goal is to use the same program in Python 2, 3 and CodeSkulptor.
Simply change
import simplegui
by
try:
    import simplegui
except ImportError:
    import SimpleGUICS2Pygame.simpleguics2pygame as simplegui
in your CodeSkulptor’s programs.

There are also some other modules usable in Python 2, 3 and CodeSkulptor. For example, simplegui_lib_loader provides a class to dealing the loading images and sounds problem in CodeSkulptor. It waits until all media are completely loaded and draw a progression bar.

Read the first page of documentation for how to install and learn few tips.
Read also the compatibility note.

Warning: If you use it to write your mini-projects, be careful, make sure they are perfectly correct in CodeSkulptor.
Only CodeSkulptor must be used to evaluate mini-projects.
