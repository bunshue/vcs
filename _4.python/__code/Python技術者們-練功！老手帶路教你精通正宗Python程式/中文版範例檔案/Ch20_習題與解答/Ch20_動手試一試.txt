### 20.2 ###

從標準函式庫中，使用datetime來管理檔案的日期/時間，
並使用os.path和os或pathlib來重命名和歸檔封存。

---

同一天出現多個檔案會出現問題。如果你有大量檔案，
存檔目錄將變得越來越難以駕馭。

---

使用「年-月-日」格式可以按日期對文字檔進行排序。
將日期放在檔名的末尾，但在副案名之前，會增加視覺
上解析日期元素的難度。

### 20.3 ###

import datetime
import pathlib
FILE_PATTERN = "*.txt"
ARCHIVE = "archive"
if __name__ == '__main__':
    
    date_string = datetime.date.today().strftime("%Y-%m-%d")
    cur_path = pathlib.Path(".")
    new_path = cur_path.joinpath(ARCHIVE, date_string)
    new_path.mkdir()                                    
    paths = cur_path.glob(FILE_PATTERN)
    for path in paths:
        path.rename(new_path.joinpath(path.name))

---

你可使用 os.path 和 os 函式庫，尤其是os.path.join()、os.mkdir() 和os.rename()。
