1. RPi目錄包含第2、3、7章Python程式與Node-RED流程，Python程式以例題命名，例如：例題3.1，檔案名稱為EX3_1.py。

2. Arduino程式分別放在各例題目錄，例如：例題4.1，目錄為EX4-1、程式名稱EX4-1.ino。

3. 第6章例題目錄中arduino_secretes.h，請填入無線分享器SSID與Password(SECRET_SSID與SECRET_OPTIONAL_PASS)
   以及在Arduino IoT Cloud取得的SECRET_DEVICE_KEY。

4. flows_examples.json為本書全部例題的Node-RED流程，可匯入使用。

5. 例題4.3、7.1會用到DS18B20，請複製DS18B20.h、DS18B20.cpp存放至arduino\libraries\DS18B20目錄。
