新华网科威特７月１８日电（记者林甦）亚奥理事会１８日发表新闻公报表示，亚奥理事会与欧洲奥委会在此间签署一项谅解备忘录，就帮助伊拉克运动员参加第十五届亚运会及２００８年北京奥运会达成一致。 
　　 
　　公报说，亚奥理事会主席谢赫·艾哈迈德·法赫德１７日和来访的欧洲奥委会主席佩斯坎特举行了会谈，双方对伊拉克奥委会主席艾哈迈德·哈吉亚上周六遭到武装分子绑架一事表示强烈关注和谴责，希望尽快释放人质，使伊拉克奥委会恢复正常工作。 
　　 
　　伊拉克奥委会不久前提出要求，鉴于伊拉克目前面临着极其动荡的安全局势，希望各方能够为伊拉克运动员提供一定的训练和比赛条件。 
　　 
　　在此次会谈中，亚奥理事会和欧洲奥委会均表示将对伊方的请求大力支持，并提出将鼓励其所属各成员给予伊拉克体育界各种可能的帮助，以使伊拉克运动员得以备战今年的多哈亚运会和２００８年北京奥运会。 
　　 
　　欧洲奥委会还表示，在即将于七月底举行的欧洲奥委会成员大会上，将同各与会成员代表进一步探论如何对伊拉克体育界进行帮助，并就相关技术性问题进行磋商。（
