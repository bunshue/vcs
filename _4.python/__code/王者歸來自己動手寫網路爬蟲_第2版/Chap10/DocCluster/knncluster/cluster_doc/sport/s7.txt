◆ 中国，“世界工厂”之“保龄工厂”，中国保龄制造业的发展机遇 
同其他许多行业一样, 国外著名保龄公司正在转向中国这个 "世界工厂" 贴牌生产自己的产品, 从保龄鞋、保龄球、零部件到成套设备； 
部分中国公司已在国际市场上也推出了自己的品牌，并取得了不俗的成绩。某公司已连续两年出口年销售额超过人民币一亿元； 
2002年中国出口的保龄产品估计超过人民币 3 亿元; 
预测今后几年中国保龄产品的出口将以超过30%以上年增长速度递增。 

◆ 国外保龄发展趋势

荧光概念 
谈及全球保龄发展趋势，只能把'荧光概念'放在第一位。从荧光球道、荧光球、荧光球瓶， 发展到荧光看板、荧光桌椅，发展到荧光球沟，甚至荧光地板！整个一个荧光球馆！ 
声像光色+DJ+快餐+酒吧 
最入流的保龄球馆，往往在两个悬挂的记分显示器中间增加一电视机，用于播放MTV等影象，配以各种舞台灯光，荧光遍洒，七彩飞扬，再加DJ助兴，男女老幼喝着各色啤酒饮料踏着舞步扔保龄球 （你疑似进了迪厅而非保龄球馆）…… 这是在欧美时尚保龄球馆夜间营业高峰时间段常常可以见到的情景。 
运动性、娱乐性、社交性， 全面抓，手手硬 
不特别强调保龄运动的某一特性, 而是争对不同消费人群的需要，制定出各种营销方式。于是有各种赛事、各种折扣、各种专场、各种奖励… 
 
中国保龄运动的起步较晚，过去只在部分大城市的为数不多的高档宾馆、酒店作为附设项目，供客人娱乐，外人很少涉足。因些被称为"贵族运动"。近几年来，随着国民物质文化水平的提高及面对大众消费群的保龄球馆的逐步增多，公众对保龄运动有了新的认识，特别是与近几年流行的卡拉OK、电子游戏和歌舞厅相比，更体会到保龄运动的高尚、健康和实惠，因而这项活动得以迅速发展。资料显示，九三年以前全国仅有十几个城市60余家场馆，计300条球道，到九五年底全国已发展到六、七十个城市，200余家场馆，总计拥有2000余条球道。到九七年六月底，已增至约3万道。 

但是，尽管中国保龄运动在短短几年内龙行虎步，可与其他发达国家和地区相比，仍存在规模数量严重不足和区域分布很不均衡的差距。目前拥有的3万余条球道相对于庞大的消费人群，远远难以满足市场需求，而从地区格局上，广东、上海、北京、江苏、浙江等地大中城市及沿海经济发达地区保龄业渐趋馆和，中西部及中小城市则尚初始发育甚至等待开发。按目前内外通行的每一个城市每十平方公里每万人一个保龄球馆的布局规则来测算，中国保龄业有较大的市场容量和广阔的前景。

保龄球运动是一项较适合中国国情的休闲运动。

在西方社会，汽车普遍进入家庭，人们的户外活动空间广阔，工作之余可选择的休闲方式很丰富，象海滨消夏、郊游、民域观光等，因此保龄球作为一种成熟的娱乐方式，属于并不火爆的"稳定消费"。而中国城市空间拥挤，人口密度大，加上汽车不普及，道路状况不好以及收入等原因，人们的生活空间相对小得多，到郊外和田园活动的机会也少得多。因而保龄球这项能够在繁华喧嚣的都市之中营造出集休闲、娱乐和运动于一体的优雅的室内活动大受宠爱。同时，其蕴含的竞技特色更符合东方口味。在一种较"严肃"的游戏规则下使大家得到沟通和交流--这种玩法一向为国人钟爱。哪怕玩的水平不高，偶然性和运气也会使您掷出好球来，这种对好运的企望使得一次次的重复变得趣味盎然；水平相当的人比一比输赢，有差距时可能通过让分较一较 
高低，一个人也可以不断提高打破自己的"记录"；无论是朋友们聚会，还是亲友之间合欢都适宜，从方式上讲，犹如一种"运动麻将"。
