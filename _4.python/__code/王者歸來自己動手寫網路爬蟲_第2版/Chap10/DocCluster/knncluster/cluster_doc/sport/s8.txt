在北京中南海保龄球队从刚刚结束的2003年全国锦标赛中传出男团五连冠捷报后，由市保协、北京卷烟厂共同举办的“中南海杯2003年北京市保龄球会员联赛等级擂台赛”，于11月6日起，在我市深圳大厦等42家会员场馆同时开战。 

　　本次联赛受“非典”疫情的影响，从6月份推迟至11月份举行，比赛的形式较往年有所精简，将以等级擂台赛的形式展开，比赛分为两个阶段，11月6日至15日为第一阶段，16日至29日为第二阶段，每阶段将从各场馆自行举办的赛事中选拔出84名选手进入复赛，复赛分6组进行1局一对一的淘汰赛，同时增加了首轮比赛负者的比赛机会，球员将依照成绩分别进入各自的区域，最后将产生本组的上、下区冠军，分别以电视挑战赛的方式与取得全国锦标赛“五连冠”的北京中南海队对垒，各组冠军将通过抽签选择对手，并可对北京中南海队的队员提出约定的限制措施，增加比赛的趣味性及保龄球爱好者的比赛经验及水平。 

比赛将于11月29日结束。
