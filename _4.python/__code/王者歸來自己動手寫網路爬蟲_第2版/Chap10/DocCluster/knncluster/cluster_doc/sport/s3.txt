“如果身体状况允许，２００８年我要到北京观看奥运会，从第一天到最后一天。相信这将是一次最成功的盛会！”年近九旬的国际奥委会终身名誉主席萨马兰奇２７日对新华社记者表示。

　　虽已８７岁高龄，但满头银发的萨翁仍然精神矍铄、面色红润。在助听器的帮助下，他不时与身边的人交流。２７日上午，当他陪同西班牙国王卡洛斯一世来到中国网通全球网管中心参观时，同样受到在场人员的热烈欢迎。

　　在中国人心中，萨马兰奇始终有着特殊重要的位置——因为他对中国人民的友好感情，因为一个西班牙籍国际奥委会主席对中国申奥的支持和帮助。早在他任国际奥委会副主席时，就协助基拉宁主席为恢复中国在国际奥委会中的合法席位做了许多工作。萨马兰奇始终坚持中国应该举办奥运会，他退休前的最大希望是奥运圣火能在北京点燃。２００１年，是他在莫斯科郑重地宣布北京获得２００８年奥运会主办权。
鉴于克里斯蒂安·齐沃与巴塞罗那好事将近，这一讯号似乎在暗示国际米兰将会把注意力转移到巴勒莫的安德烈·巴尔扎利身上。

据意大利国内的媒体爆料，国际米兰的官员们已经同巴勒莫俱乐部就将这位意大利国脚带到梅阿查的可能性进行了探讨。

在经历了三次令人愉快的国家队集训体验后，巴尔扎利更坚定了要加入一支意大利豪门的决心，并且他与马尔科·马特拉齐有着非常不错的私交。上个赛季关于这名26岁的后卫与尤文图斯的绯闻一直报道不断，甚至有传言双方俱乐部都已经达成协议，只是未最终签订合同。

如果齐沃最终签约巴塞罗那，可以预见，国际米兰的官员将会立马为这位托斯卡纳人准备一份合约，巴尔扎利被认为是国家队中卡纳瓦罗的接班人。

 与此同时，有报道称阿德里亚诺在巴西度假的时候，受到了不明身份者的恐吓敲诈。巴西国内某不知名媒体称有两名男子接触了巴西人的经纪人，威胁这位目前近况不佳的球星，想要消灾必须破财！随着大卫·苏亚佐的到来，阿德里亚诺很有可能被用来作为在与皇马交换马哈马多·迪拉亚的筹码。
