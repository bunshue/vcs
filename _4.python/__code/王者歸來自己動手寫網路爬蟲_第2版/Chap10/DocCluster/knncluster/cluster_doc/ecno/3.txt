中新社乌鲁木齐七月十四日电 题：中央企业“群雄逐鹿” 百亿资本涌入新疆

　　中新社记者 陶拴科

　　近年来，新疆依托丰富的能源、水土光热资源、矿产资源优势和毗邻中亚的地缘优势，吸引了大批中央企业纷纷“抢滩”新疆市场。 

　　目前，在新疆投资的中央企业有中石油、中石化、中粮集团、中材集团、中国电力投资集团、神华集团、国家投资公司、徐矿集团等中央大型企业数十余家，投资资金已达数百亿元人民币。

　　记者了解到，在新疆边远的塔城地区，借助国务院国资委及中央企业援疆的机遇，依托援疆官员的牵线搭桥，智力支持，截至目前，落户塔城的中央企业已有十余家，涉及到能源、矿产、农产品等领域，投资规模突破三十亿元。 

　　资料显示，新疆目前已发现各类矿产一百三十八种，占中国已知矿种的百分之八十。石油远景资源量二百零八亿吨；天然气十万亿立方米；煤二点一九万亿吨；铁、铜、镍、金等重要矿产资源储量丰富。具备建成中国重要的石油、天然气化工基地、煤电化工基地和有色金属等重要战略资源接替区。

　　新疆除资源丰富之外，还有十七个一类口岸，各大口岸在边境线上形成了一条长长的商贸走廊，通过这些星罗棋布的开放口岸，促使中国东部省区的工程机械、汽车、化工产品、食品、纺织品源源不断地进入中亚诸国，有的甚至从此通道远销欧洲，商贸往来频繁。

　　据业内人士分析，随着西部大开发向纵深发展，西部已经成为中国投资的热土，中央企业大举西进是必然趋势，西部地区经济的快速发展，又为企业“资本西进”提供了可靠的前提和保障。在中国能源建设“梯队”中，新疆正逐渐由“后备军”转变成“主力军”。 

　　新疆已进入西部大开发以来资金汇聚最为集中的时期，资本西进渐成气候，数百亿资金西涌逐利而来。目前仅国务院国资委中央企业“西进”资本已有五百亿元之多。(完)
