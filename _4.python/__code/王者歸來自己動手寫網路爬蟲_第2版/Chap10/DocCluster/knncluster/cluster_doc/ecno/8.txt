　基本面上，中國人民銀行金融統計資料顯示，二季度我國外匯儲備增長1305.9億美元。 截至6月末，國家外匯儲備餘額為13326億美元，同比增長41.6%。居民儲蓄存款餘額達169539.59億元。6月份廣義貨幣供應量（M2）、狹義貨幣供應量（M1）、新增貸款都有所反彈，表明貨幣信貸仍處於強勁增長態勢。由於受到股市波動影響，6月份居民儲蓄存款扭轉過去兩個月下降的局面，同比多增136億元。

　　市場普遍認為，貨幣信貸強勁增長，表明前一階段採取的緊縮措施效果不明顯，短期內減免利息稅、加息、提昇準備金率、特別國債、央行票據的發行等收縮流動性的措施都有可能出台。但是最近新基金密集發行，表示管理層對大碟一定程度的呵護。

　　國際市場，穆迪和標準普爾兩大評級機構的"降級"宣告，幾乎成了壓破美國次級抵押貸款泡沫的"最後一根稻草"。這樣的舉動也反映出美國次級抵押貸款市場狀況的進一步惡化，伴隨著美國樓市降溫，次級抵押貸款這一高風險領域的危機也逐漸顯現。次貸風波對美國金融市場以及總體經濟的衝擊，也在匯市得到表現，由於投資人預期美國經濟將持續低迷，且美聯儲可能提早轉入降息，美元兌歐元、英鎊甚至日元都出現大幅下挫，兌歐元則再次更新歷史低點。美元指數在7月12日創下80.55的低點,離2004年12月30日創下的80.42的低點僅一步之遙。由於美元指數並未考慮到人民幣的升值的因素，如考慮的話,美元指數應該已經跌穿這一低價。美元的另一個低點是1995年4月28日創下的80.05。可見，美元指數已經接近12年來的低點。與此同時，此前公佈的零售銷售資料鞏固了投資者對消費者支出的信心，力拓斥資約380億美元收購加拿大鋁業引發更多併購活動接踵而至的預期，道指283點的漲幅再次更新歷史紀錄，也擊破了前期形成的似乎"三重頂"的走勢。恆指繼續大幅上揚，原油價格再次回到70美元以上，銅接近8000美元，部分農產品價格堅挺，預示全球通貨膨脹壓力加大。

　　從仿真交易的市場情況來看，合約月份之間的價差由原來的動輒300到500點，逐步縮小為目前的200點，有的已經不足100點，遠期12月的預期價格也由原來預期達5500點以上，迅速降為目前的4500點左右。表明市場普遍的樂觀預期和狂熱氣氛已經發生較大的改變。從週五的收碟情況看，遠期期指也只是暫時止跌而已。

　　技術上，自上週五下午多頭的絕地反擊以來，本週五個交易日上證綜指基本維持小幅度的整理。從均線上來看，指數雖然已經得到短期均線的支撐，但仍然受到20日和60日均線的壓制。成交量的萎縮相當明顯，證明市場暫時處於觀望狀態，或暫時的平衡。而且由於印花稅的調高，對於短線操作的遏制是明顯的，並不像有些人分析的，地量就表示將要下跌。
