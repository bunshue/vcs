
===============================================
關於範例檔案的使用說明
===============================================

●關於範例檔
範例檔的內容包含《Python資料可視化攻略》（978-4-7981-6397-0）內文解說的程式碼與範例程式的執行結果（從第4章到第8章、附錄）、CSV檔案、以及出版社提供的的原創圖檔。
範例檔為Jupyter Notebook格式。

●本書範例程式執行環境
本書的範例已於Anaconda（Windows）的環境下經過測試。
可在下列的環境下正常運作。

・OS：OS Windows10（64位元版）

．網頁瀏覽器：Google Chrome（只有第6章使用FireFox）
