　　张雄细说五银


　　
　　本报记者薛原李增辉


　　“既可惜，也不可惜。”中国游泳队今天拿下五枚
银牌后，总教练张雄这样评价道：“说可惜，因为在上
午的预赛中，3 个项目中国队都有两人闯入决赛。当时
大家想，总有一个项目可以拿到金牌吧。谁想到3 个项
目都没有保住。说不可惜，因为今天的五个项目中国队
原本就处于劣势。赛前就预料到这也许是我们最‘黑’
的一天。现在看，情况还比预料的要好一点点。”


　　今天的五个项目，中国队除了男子4 0 0 米混合泳
由老将熊国鸣担纲外，其余的全是新人出阵。第一项男
子1 0 0 米蝶泳，日本的山本贵司今年的最好成绩是5 
3 秒3 9 ，今天又游出了5 3 秒3 4 的成绩，创下了新
的亚洲纪录。从实力讲，中国队的张宵和张强同他确实
有差距。第二项女子2 0 0 米自由泳，昨天覃彩妮在2 
0 0 米接力中游出了2 分0 秒7 3 的个人最好成绩，如
果今天能保持这股势头，金牌就是她的了。可惜覃彩妮
是第一次参加国际大赛，同中国台北队的名将蔡淑敏比
起来经验差很多。而且，她的心理压力也比较大，最后
导致失利。


　　第三个项目男子4 0 0 米混合泳，中国队的老将熊
国鸣游得相当出色。但是日本队的森隆弘年轻气盛，最
后在体力上占据优势而获胜。第四项更可惜，要论速度
，中国队的小将齐晖其实比日本的田中雅美要快，输就
输在最后一个转身上，说到底还是技术不过关。最后一
项男子4 ×2 0 0 米自由泳接力，中国队的第二棒杜杰
游的稍差，失利在所难免。


　　“不过。”张雄话锋一转，“从这两天的比赛看，
日本队并非赛前预料的那么可怕。在一些我们本以为无
望的项目上，实际上中日两队展开了激烈的争夺。昨天
晚上团部开会，我们确定了新的作战思想，那就是甩开
今年已有的成绩依据，一项项地和日本队去拼。我们要
抛掉自己是亚洲游泳强国的老观念，将自己摆在拼的位
置上去。”


　　“从这届亚运会看，中日两国都将目标放在了2 0 
0 0 年的悉尼奥运会上。亚运会我们派出这么多新人出
战，难免有闪失。中日两国在游泳项目上的竞争会持续
很长一段时间。”


　　（本报曼谷1 2 月8 日电）
